1. Create 3 user accounts with different roles,
Admin - admin@eventjini.com / eventjini
Customer - customer@eventjini.com / eventjini
Supplier - supplier@eventjini.com / eventjini

2. When you login as Admin, "User Role Settings" menu link in the left side menu. You can be set which user role can access which menu pages.

3. When you login as Customer or Supplier, left side menus are displayed based on their user role pages assigned by Admin.
